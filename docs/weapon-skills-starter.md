# Weapon Skills (Starter)

All of these are **starter skills** available when the character first uses the weapon category.

**Total: 170 skills** (10 per weapon category × 17 categories)

## Implementation Notes

### Database Schema

```prisma
model Skill {
  // ... existing fields
  tier          String  @default("Starter")  // "Starter", "Advanced", "Expert", "Master"
  categoryId    String?  // null = universal skill
  category      WeaponCategory? @relation(...)
}
```

### Skill Learning Rules

- **Starter skills** are automatically available when equipping a weapon category for the first time
- Players can learn all 10 starter skills for free from Skill Trainer NPC
- Advanced/Expert/Master skills require:
  - Higher player level
  - Weapon proficiency (usage count)
  - Quest completion or rare drops
